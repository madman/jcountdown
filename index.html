Simple countdown as JQuery Plugin

Author: madman

Source: on Github

Download: latest minified

Examples

Simple countdown

HTML: 

<div id="countdown-1" data-time="3600"></div>

Java Scripts: 

$(function() {
	$('#countdown-1').countdown();
});

Multi countdowns

HTML: 

<div class="countdown-2" data-time="10"></div>
<div class="countdown-2" data-time="60"></div>
<div class="countdown-2" data-time="600"></div>
<div class="countdown-2" data-time="3600"></div>

Java Scripts: 

$(function() {
	$('.countdown-2').countdown();
});

Countdowns with callback

00:00:10 Run

HTML: 

<span id="countdown-3" data-time="10" data-text="The end!">00:00:10</span>
<a id="run" href="javascript:;">Run</a>

Java Scripts: 

$(function() {
	$('#run').click(function() {
		$('#countdown-3').countdown(function(timer) {
			alert(timer.data('text'));
		});
	});	
});
